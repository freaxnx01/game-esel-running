# Esel vs. Karotte 🐴🥕

Ein lokales 2-Spieler-Browserspiel: *Esel vs. Karotte – Das große Rennen*. Zwei Esel jagen der Karotte hinterher – wer sammelt mehr Punkte?

## ▶️ Spielen

**[Hier spielen → https://github.freaxnx01.ch/game-esel-running/](https://github.freaxnx01.ch/game-esel-running/)**

Läuft komplett im Browser – keine Installation nötig.

## Steuerung

| Aktion | Taste |
|--------|-------|
| Spieler 1 (oben) | Leertaste (Space) |
| Spieler 2 (unten) | Enter |
| Pause | `P` |
| Neustart | `R` |

## Lokal ausführen

Entweder `index.html` direkt im Browser öffnen, oder aus dem Repo-Root servieren:

```sh
python3 -m http.server 8000
```

dann <http://localhost:8000/> aufrufen.

## Deployment

Bereitgestellt über **GitHub Pages** vom `main`-Branch (Root), erreichbar unter der Domain `github.freaxnx01.ch`.
